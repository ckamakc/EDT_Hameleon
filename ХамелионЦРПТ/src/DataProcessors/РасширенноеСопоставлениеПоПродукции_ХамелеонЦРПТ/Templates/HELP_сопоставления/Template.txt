Данная обработка позволяет проставить соответствия
номенклатуре в 1с и продукции из ЦРПТ.
Очень удобно использовать фильтр показать не сопоставленную.
+для конфигураций хранящих остатки на регистрах накопления,
можно также сопоставить и остатки 1с с остатками в ЦРПТ.
(Для других конфигураций хранящих остатки не в регистрах, также планируется дописать сопоставление остатков, если необходимо обращайтесь)

При первоначальном использовании 
перейдите на вкладку настройки и выберите регистр остатков из 1с
с которым планируете сверять остаток ЦРПТ,
также можно использовать отбор нажав кнопку(Обновить список)
Обязательно отметься реквизиты для анализа
слева-номенклатура у выбранного регистра остатков
справа-это обычно количество, для сверки с количеством в ЦРПТ.

Можно сопоставить сразу несколько элементов продукции ЦРПТ
с одним наименованием 1с-для этого выделите нужные строки и нажмите сопоставить
в открывшемся списке выберите нужный элемент для сопоставления.

Также при первом открытии на вкладке сопоставление может потребоваться задать поле
Справочник из 1с=Справочник.Номенклатура

Если сопоставляется несколько элементов ЦРПТ с 1ой номенклатурой из 1с, то
в справочник продукция в поле реквизит для сопоставления записывается код номенклатуры 1с,
таким образом удается задать соответствие нескольких элементов к одному.
